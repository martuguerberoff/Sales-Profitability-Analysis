# 📊 Sales-Profitability-Analysis

Análisis de ventas y rentabilidad utilizando SQL y Power BI, incluyendo modelado de datos, procesos ETL y visualización de KPIs.
Proyecto académico | SoyHenry | 2026

👥 Realizado por Martina Guerberoff


## 🎯 Objetivo general del proyecto

Desarrollar un modelo analítico integral que permita evaluar el rendimiento comercial y la rentabilidad del negocio, facilitando la identificación de oportunidades de mejora y la toma de decisiones estratégicas basadas en datos.


## 🛡️ Objetivos específicos

1. Integrar y transformar los datos provenientes de distintas fuentes en un modelo unificado y consistente.
2. Analizar el rendimiento de ventas considerando variables como ingresos, costos y márgenes de ganancia.
3. Desarrollar métricas clave (KPIs) que permitan monitorear la performance del negocio.
4. Diseñar un dashboard interactivo que facilite la visualización de los resultados y el análisis de la información.
5. Generar conclusiones que contribuyan a la optimización de la rentabilidad y la toma de decisiones estratégicas.


## 🚀 Alcance del proyecto

* **Ingeniería de Datos (ETL):** Extracción, transformación y limpieza de datos utilizando SQL y Power Query para asegurar la calidad y consistencia de la información.
* **Modelado de Datos:** Construcción de un modelo relacional optimizado para el análisis, incluyendo tablas de hechos y dimensiones relevantes.
* **Análisis Exploratorio (EDA):** Evaluación del comportamiento de ventas, costos y rentabilidad mediante métricas descriptivas.
* **Desarrollo Analítico:** Creación de medidas y columnas calculadas utilizando DAX para representar lógicas de negocio.
* **Visualización Ejecutiva:** Diseño de un dashboard interactivo en Power BI para el análisis de KPIs y soporte a la toma de decisiones.


## ⚙️ Stack Tecnológico

* **Motor de Base de Datos:** SQL Server (gestión y consulta de la base de datos).
* **Lenguaje de Consulta:** SQL (extracción y manipulación de datos).
* **ETL y Preparación de Datos:** Power Query (limpieza, transformación y estandarización de datos).
* **Modelado y Cálculos Analíticos:** DAX (creación de métricas, KPIs y lógica de negocio).
* **Visualización y Business Intelligence:** Power BI (desarrollo del dashboard interactivo y análisis visual).


## 🧠 Principales hallazgos

* Identificación de productos con alta participación en ventas pero baja rentabilidad, lo que evidencia oportunidades de optimización de precios o costos.
* Detección de variaciones significativas en la rentabilidad según categoría y segmento de cliente.
* Análisis de tendencias temporales que permiten identificar períodos de mayor y menor desempeño comercial.
* Identificación de KPIs clave para el monitoreo continuo del negocio (ingresos, margen, volumen de ventas).


## 📊 KPIs analizados

* Ingresos totales
* Costos totales
* Margen de ganancia
* Rentabilidad por producto
* Volumen de ventas
* Performance por categoría y período


## 📌 Conclusiones

* La integración de datos y el modelado analítico permiten obtener una visión integral del negocio.
* El uso de dashboards interactivos facilita la identificación de oportunidades de mejora en la rentabilidad.
* La definición de KPIs claros permite monitorear el desempeño y tomar decisiones basadas en evidencia.

---

## 💡 Recomendaciones
* Optimizar la estrategia de precios en productos con alta demanda pero bajo margen.
* Focalizar esfuerzos comerciales en categorías con mayor rentabilidad.
* Implementar un seguimiento continuo de KPIs para mejorar la toma de decisiones.
